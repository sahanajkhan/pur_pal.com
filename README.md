## 🌟 Features

* 📢 Centralized Announcement System

* 💬 Doubt Solving Forum
* 📅 Smart Schedule Management
* 🔔 Real-time Notifications
* 🤖 AI-powered assistant *(coming soon)*

## 🏗️ Architecture

Client → Server → Database → External Services (AI / Notifications)

---

## 🛠️ Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Other Tools**

* JWT Authentication
* Socket.io (planned)
* Firebase (optional)

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/campus-connect.git
cd campus-connect
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create `.env` in `/server`

```
PORT=5000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_secret
```
